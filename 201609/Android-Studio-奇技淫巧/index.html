Android Studio 奇技淫巧

  • 插件推荐
  • Logcat配色
  • 快捷键

奇技淫巧：指过于奇巧而无益的技艺与制品。
貌似在这里使用不太好，毕竟“无益”有点说不过去了。
Android Studio版本为2.2 Surface Book Win10专业版（其实蛮好用的，至少在SB上蛮好用）

插件推荐

  • ADB WIFI
    局域网内无需数据线的ADB调试工具，本人所用电脑为Surface Book，本来USB接口就只有两个，及其不够用，又穷的没钱买Dock，所以这个插件对我而言就极好极好的。（其实我是看着桌上一堆线烦）
    Android wifi无线调试App新玩法ADB WIFI
  • Android ButterKnife Zelezny
    结合ButterKnife使用效果更佳哟。你还在使用findViewById，setOnClickListener…这些玩意嘛，快来换换手感试试ButterKnife，配合插件自动绑定Id，代码一下子清爽了好多。
    用法 记得一定要在”R.layout.**“上点右键选择Generator（windows下快捷键Alt+Insert）
  • Android Material Design Icon Generator
    Module或者Project文件夹上右键new的最下面，这个就不多说了
  • 主题插件 Material Theme UI
    装上之后记得去设置里面Setting->Editor->Colors&Fonts->General里面选择一个theme，至于更改字体什么的就不多说了。
  • 有关插件必须要说的，请注意
    虽然插件大法好，但是可不要贪杯哟。本来Android Studio已经机器耗内存了，你再装那么多插件不怕爆炸吗，不常使用的就关掉吧，以下时我自己的插件开关情况。


Android Logcat 配色方案

都是设置foreground

  • Assert #FF4949
  • Debug #6897BB
  • Error #FF6B68
  • Info #6A8759
  • Verbose #BBBBBB
  • Warning #BBB529
